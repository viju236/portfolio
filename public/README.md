# Portfolio Media Assets

This folder contains images and videos for your portfolio website.

## Folder Structure

```
public/
├── hobbies/
│   ├── badminton1.jpg      # Badminton action shots
│   ├── badminton2.jpg
│   ├── badminton-match.mp4 # Video highlights
│   ├── music1.jpg          # Music performances
│   └── performance.mp4     # Live performance video
└── wedding/
    ├── bts1.jpg            # Behind-the-scenes wedding prep
    ├── venue.jpg           # Venue photos
    ├── shopping.jpg        # Wedding shopping moments
    └── engagement.jpg      # Engagement ceremony photos
```

## How to Add Your Media

1. **Take or collect your photos/videos**
2. **Rename them according to the structure above**
3. **Place them in the appropriate folders**
4. **The website will automatically display them!**

## Supported Formats

- **Images:** JPG, PNG, WebP
- **Videos:** MP4, WebM

## Recommended Sizes

- **Images:** 1920x1080px or 1200x800px
- **Videos:** 1080p (1920x1080) or 720p (1280x720)

## Tips

- Keep file sizes reasonable (under 5MB for images, under 50MB for videos)
- Use descriptive filenames
- Compress images before uploading for faster load times
